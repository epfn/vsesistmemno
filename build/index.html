Разработка электронных каталогов

Управление информационными данными компании

Любые не системные документы, информацию, приводим в системную структуру, каталогизируем, разрабатываем документы в едином стиле, вносим в базы данных компании и сервисы, где эта информация начинает работать.

Для чего необходим сервис

  • Любая компания, в какой-то момент сталкивается с проблемой управления и хранением рабочей документации сотрудников для надлежащего исполнения ими своих должностных обязанностей.
  • Для ввода документации в информационные ресурсы, ее сопровождение, контроль за актуализацией требуются руководители и сотрудники с высокой организационной мотивацией и системных подходом для выполнения такого рода работы. В том числе такая работа является для компаний дополнительной нагрузкой на сотрудников.
  • Компания Всё системно состоит из специалистов высокого уровня по разработке и систематизации данных, и эту часть работы берет на себя.

Состав документации:

Группа общие правила работы организации

  • Правила и требования работы в организации и подразделениях
  • Правила безопасности труда
  • Телефонные и контактные справочники

Группа документации по правилам работы сотрудников

  • Персональные офферы сотрудников при приеме на работу.
  • Должностные инструкции
  • Регламенты работы сотрудников на конкретных рабочих участках
  • Персональные требования к сотрудникам
  • Протоколы собраний, закрепление договоренностей

Группа производственной документации

  • Инструкции по выполнению производственных операций
  • Календарь повторяющихся событий, дни рождения сотрудников, календарь ТОИР обслуживания оборудования и сервисов и др.

Группа информационных данных

  • Базы данных ресурсов компании.Пример, архив выполненных работ (описание, фотоотчет), база объектов и т.п.

Группа договора с контрагентами

  • Договора, приложения.

О системе
  • Подходит для создания и работы базы знаний.
  • Для протоколирования совещаний и фиксации постановлений.
  • Должностные обязанности и инструкции, персонализация офферов (должностные обязанности, условия прохождения испытательного срока) сотрудников.
  • Контроль за календарем повторяющихся процессов. События по днями годам.
  • Для учащихся и образовательных организаций: конспекты учащихся, билеты для подготовки к экзаменам.
  • Создание инструкций в красивом фирменном стиле.
Система имеет виды доступа
  • Персональный пользователя – для личных данных.
  • Руководитель отдела кадров, руководитель отдела - персональные данные сотрудника, требования к сотруднику отдела – должностные инструкции, цели, оффер.
  • Раздел отдела – общая документация отдела (регламенты, инструкции, должностные обязанности).
  • Раздел компании – общая информация компании.

Разработка электронных каталогов

  • Разработка электронных каталогов: информация, документы, объекты и оборудование учета, справочники.
  • Отбор данных (фильтры, регистры учета, теги, характеристики) в различных разрезах учета.
  • Электронные книги в виде статей по разделам.
  • К каждой сущности элемента учета к информации можно привязать общие группы, объединяющие характеристики, что позволяет существенно улучшить поиск и работу с информацией.
  • Электронные справочники по отраслям: пример, медицинские справочники.

Разработка электронных каталогов: Арт направление

  • Создание электронных каталогов произведений искусств.
  • Персональные коллекции, федеральные каталоги по различным тематикам

Компания Все системно разработала и запустила федеральный проект электронный музей для публичного пользования - электронный филателистический каталог почтовых марок vcstamps.ru

В каталоге более 10.000 элементов, которые структурированы по темам, разделам, имеют мощный инструмент фильтрации для отбора данных.

Проект поддерживается и постоянно актуализируется.
